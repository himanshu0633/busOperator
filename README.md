# busOperator

## GitHub Pages pe live kaise karein

1. Is repo ko GitHub par push karein.
2. `main` branch par push hote hi workflow `Deploy to GitHub Pages` automatically chalega.
3. GitHub repo me **Settings → Pages** par jaake source ko **GitHub Actions** select karein.
4. Deployment complete hone ke baad aapko live URL mil jayega:
   `https://<your-username>.github.io/<repo-name>/`

> App me `HashRouter` use kiya gaya hai, isliye deep-link refresh issue nahi aayega on GitHub Pages.
